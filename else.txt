elseを使用する時の注意点
elseの使用はifがある時だけ使用できるので独立して使用はできない。


JavaScriptで実装した時の確認ポイント
ifとelseの間にconsole.logを入れる場合注意していれる。
コンパイルエラーに注意する。

数値型を比較してelseの処理に入る処理を作ってくださ


const num1 = 10;
const num2 = 20;
if (num1 > num2) {
    console.log("num1がでかい");
} else {
    console.log("num2がでかい");
}

文字列型を比較してelseの処理に入る処理を作ってください。

const str1 = "nasi"
const str2 = "apple"

if (str1 === str2){
console.log("梨とりんごは同じ")
} else {
    console.log("梨とりんごは違う")
}
